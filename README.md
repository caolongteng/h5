## 前端工作流
### 版本控制工具
    git
### 开发工具
    编辑器：webstorm，sublime  
    调试工具：chrome(一般H5)，微信开发者工具（公众号或小程序，依赖openUdid）  
### 开发规范
    Css：Sass预编译  
    Js：ES6规范，模块间的依赖依靠import、export
    静态资源文件：做一些压缩优化处理
### 框架\类库
    Vue、React、ReactNative等数据驱动框架
    Jquery、Zepto等dom操作库
    其他依赖于业务的组件或类库
### 项目构建工具
    gulp构建，辅以webpack打包处理js
    npm管理依赖
    nvm管理npm
### 抓包工具
    http抓包：charles
    socks抓包：Wireshark
